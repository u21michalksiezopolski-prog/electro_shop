# Electro Shop - Aplikacja E-commerce

Aplikacja webowa sklepu elektronicznego w **PHP**.

## Wymagania

- PHP 7.4+ 
- MySQL 5.7+ lub MariaDB
- Serwer web lub XAMPP

## Konta testowe

- **Admin:** `admin@electroshop.pl` / `123`
- **Pracownik:** `pracownik@electroshop.pl` / `123`
- **Klient:** `customer@electroshop.pl` / `123`

## Funkcjonalności

✅ Przeglądanie produktów z filtrowaniem  
✅ Koszyk zakupów (działa bez rejestracji)  
✅ Rejestracja i logowanie  
✅ Reset hasła  
✅ Składanie zamówień  
✅ Historia zamówień  
✅ Profil użytkownika  
✅ Lista ulubionych  
✅ Panel administracyjny
